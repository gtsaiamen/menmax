Формула Керна
1. Точка зрения
Конкретное утверждение по теме поста
2. История с эмоцией
Личная или кейс клиента — реальная
3. Метафора
Понятный всем образ для закрепления
4. Мягкий CTA
Без давления, от истинного "я"
Особенности MAX
• Только нативные посты, без кабинета
• Форматирование как в ТГ — жирный, курсив, ссылки
• Обязательна маркировка «Реклама» + erid токен
• Меньше конкурентов → выше CTR
⚠️ Маркировка обязательна по закону с 2023 года. Получить erid через ОРД (Ozon ОРД, МТС Маркировка и др.)
